Privacy Policy for Simple.biz Call Coach

Last Updated: April 8, 2026

1. Introduction

Simple.biz Call Coach is a Chrome extension developed by the Simple.biz AI & Automation Team. It provides real-time AI-powered coaching during live sales calls by capturing and analyzing call audio through the browser. This extension is intended exclusively for internal use by Simple.biz team members.

2. Information We Collect

When you use Simple.biz Call Coach, the extension may collect the following:

  • Call Audio: Audio from active call tabs is captured in real time for transcription and AI coaching analysis.
  • Call Session Data: Metadata about coaching sessions including timestamps, call duration, and coaching scores.
  • User Activity: In-extension interactions such as session start/stop events and coaching feedback responses.

3. How We Use This Information

  • To provide real-time AI-powered coaching suggestions during live sales calls.
  • To generate post-call summaries and performance scores for individual reps.
  • To improve the accuracy and relevance of coaching recommendations over time.

4. Data Storage and Access

Call audio is transmitted securely to the Simple.biz AWS Lambda backend (wss://wu4pgdpdv9.execute-api.us-east-1.amazonaws.com/production) for processing. Session data and preferences are stored locally in the browser using Chrome's storage API. No data is stored permanently on external servers beyond what is required for real-time processing.

5. Information Sharing and Disclosure

Data collected by Simple.biz Call Coach is used solely within the Simple.biz organization. We do not sell, trade, or transfer any user data to third parties. Data is not used for any purpose unrelated to call coaching.

6. Data Security

All data transmitted between the extension and the backend is encrypted using secure WebSocket (WSS) connections. Access to backend systems is restricted to authorized Simple.biz personnel only.

7. Changes to This Privacy Policy

We may update this Privacy Policy from time to time. Any changes will be reflected on this page with an updated "Last Updated" date. Continued use of the extension after changes constitutes acceptance of the revised policy.
